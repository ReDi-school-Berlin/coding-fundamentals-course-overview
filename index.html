Introduction

Python is a high-level, general purpose programming language with serveral advantages:

  • Simple syntax → easy to learn & quick to write in
  • Versatile
  • Beginner friendly
  • Open source → free to use and distribute
  • Lot of code already written that can be reused (modules & libraries)
  • Large and active community

Variables

Variables are containers for storing data. A variable is created when you assign a value to the name of a variable (e.g. year = 2021). You can change the value of variables later on.

The naming convention for variables in Python is that all variables should be written in "lower_snake_case" (all letters in lowercase and words separated by an underscore) and that they cannot start with a number (but they can contain numbers).

There are different types of variables:

Integer (whole numbers): You can create integers just by typing a number:
my_favourite_number = 4 Float (decimal numbers): Use a dot for the decimal point:value_of_pi = 3.14String (words or sentences): You need to wrap strings either with single or double quotes. It doesn’t matter which one you use, just be consistent!school = 'ReDi' course = 'Intro to Programming' motto = "Don't worry, be happy!"As you can see in the last example above, you might have to combine double and single quotes in a single string: If you need a single quote inside the string use double quotes to wrap it - and vice versa.

You can create multi-line strings with three double or three single quotes.'''This is a multi-line string and you can use either single or double quotation signs to wrap it. It's up to you!''' Boolean (True or False): This variable type represents one of two values: True or False Booleans have to be written with capital first letters (True, not true). You can assign a Boolean value to a variable.is_redi_school_awesome = True print(is_redi_school_awesome)  → will print True Furthermore, when you compare two values, the expression is evaluated and Python returns the Boolean answer:print(10 == 10)  → True print(30 != 30)  → False print('a' == 'a')  → True print('a' == 'b')  → False

You can check which data type is given by using the type-function and you can also converse one data type into another:print(type(True))  → class 'bool' print(type(5 > 3))  → class 'bool' print(bool(0))  → False print(bool(1))  → True float("5.2")  → turning string into float int("8")  → turning string into integer int(9.2)  → turning float into integer str(3.14)  → turning float into string

Operators

Operators are used to perform operations on variables and values.

Arithmetic Operators perform common mathematical operations:5 + 5  → Addition 150 - 492  → Subtraction 5 * 4  → Multiplication 80 / 4  → Division Additionally:5 ** 5  → exponent (raise to power) 20 % 3  → modulo (remainder of division)

Assignment Operators assign values to variables:var = 5  → assign var += 5  → add and assign var -= 5  → subtract and assign var *= 5  → multiply and assign var /= 5  → divide and assign

Comparison Operators compare values, sometimes they are called relational operators. Please note: you need TWO equal signs to make a comparision; a single equal sign is used to assign a value to a variable.10 == 10  → equal to 30 != 31  → not equal to 5 < 90  → less than 45 <=46  → less than or equal to 89 > 67  → greater than 65 >= 34  → greater than or equal to

Identity Operators compare the objects, not if they are equal but if they are actually the same object with the same memory location. "var1 is var2" returns True if both variables are the very same object and "var1 is not var2" returns True if both variables are different objects. Example:x = ["apple", "banana"] y = ["apple", "banana"] z = x print(x is z) → True: x is the same object as z print(x is y) → False: x is not the same object as y, even if they have the same value print(x == y) → True: x is equal to y. This is the difference between the operators "is" and "==".

Logical Operators (and/or/not) are used to combine conditional statements:x > 5 and x < 10 → returns True if both statements are true x < 5 or x < 4 → returns True if one of the statements is true not(x > 5 and x < 10) → reverses the result as it returns False if the result of the statement inside the brackets is True

Conditions

Conditional statements check the value of an expression and provide different actions based on its outcome:

  • if determines the execution of code depending on the evaluation of a boolean expression. Is this given condition satisfied: Yes or No? If Yes do this.
  • elif allows for continued checks to be performed after the initial "if"-statement. You can add as many elif-statements as you want. Is maybe this condition satisfied? Yes or No? If Yes do this.
  • else provides an alternative code to be executed when the previous "if"-statement   - and all other optional elif-statements  - evaluate to False.
Pay attention to the colon and indentation in the next line following each condition: if occupation == "government employee":   print("You can apply for benefits.") elif occupation == "entrepreneur":   print("You make sure you pay your tax!") else:   print("This facility is only for government employees!")You can combine different conditional statements or even insert further conditions inside a given if-/elif-statement:if occupation == "government employee" and number_of_children > 0:   if number_of_children == 3:     print("You can only claim benefits for up to two children.") elif occupation == "entrepreneur":   print("You make sure you pay your tax!") else:   print("This facility is only for government employees!")

Loops

Loops iterate over a given sequence repeatedly until a given condition is satisfied.

For Loop
A For Loop iterates over the individual elements of the objects you feed it. Its syntax is structured as follows: "for variable in iterable: do something with variable"for letter in 'hello':   print(letter) for i in 'hello':   print(i)As you can see above, it does not matter how you call the respective variable; both "letter" and "i" work.

You can use a for-loop also with the range-function in order do iterate over a certain range of numbers. You always need to provide a limit to the range (starting point is automatically 0) but you can optionally also define an alternative start and also a step: range(start, stop, steps). Please note: the given ending number is excluded.for i in range(1, 100):   print(i)  → printing numbers from 1 to 99 for i in range(10, 0, -1):   print(i)  → printing numbers from 10 to 1, decreasing for i in range(10, 0, -2):   print(i)  → printing every second number from 10 to 2, decreasing You can nest a loop inside another loop. For each iteration of the outer loop the inner loop re-starts and completes its execution before the outer loop can continue to its next iteration.for i in range(1,6):   for j in range(0,i):     print("*", end=' ')   print(" ") * * * * * * * * * * * * * * *

While Loop
As long as the expression that follows a while statement evaluates to True, the block inside the while-loop will be executed repeatedly. In this example only odd numbers below 100 are printed (the loop will break as soon as the variable number is equal to 100 and the statement evaluates to False):number = 0
while number < 100:   if number % 2 !=0:     print(my_number)     number +=1You can create an infinite loop as follows:while True:Helpful commands or "magic words" for loops are:

  • break: causes a break out of the respective loop
  • continue: to start with the next iteration (at the very beginning of the loop)
  • pass: will not do anything  - which is helpful while coding a loop and trying to run it without errors even if some conditions are not yet specified.

Functions

A function is a block of code that runs when it’s called. Functions are helpful to reduce duplicate code as they are reusable and they make code easier to read.

Syntax: The word "def" lets python know, you are about to define a function. The function_name uniquely identifies the function and is followed by a pair of parentheses and a colon "():". The function body itself is written below with indentation.def function_name():   indented function body   indented function body function_name()  → to call the function type the function name followed by parenthesis

You can optionally pass certain data, called parameters, into a function. Parameters are specified after the function name, inside the parentheses. You can add as many parameters as you want, just separate them with a comma.def multiple_of_five(num):   if num % 5 == 0:      print(f"{num} is a multiple of five.")   else:      print(f"{num} is not a multiple of five.") multiple_of_five(45)   → calling the function with the argument 45 (value of parameter num)

The Python interpreter has a number of built-in functions; only some of them will be presented throughout this website, e.g.:

len() gives the length of an object or a collection of data (works with lists and strings).print(len("hello"))  → 5

Another possibility is to import functions: You can either import a whole module using "modul.function()" to call the respective function:import random print(random.randint(0,100)) print(random.random())Or you can import only the function you need from a module using directly function() to call it afterwards  - use a comma when importing various functions simultaneoulsy:from random import randrange,choices print(randrange(2,21,2))  → returns a random number between 2 and 20, skipping the odd numbers (sytax: randrange(start, stop, step)) list = [0, 1] print(choices(list, weights=(25, 75), k=1))  → returns 0 or 1 but with the probablity of 1:3 (k is optional: you can choose how many random outputs you want returned)

Formatting Code

Commenting code means adding descriptions of how pieces of your code works. You can add a single-line comment with the hash symbol # or a multi-line comment wrapped in triple-quotes '''. Lines which are thus 'commented out' are ignored by Python.# This is a comment in Python

Linebreaks inside a print statement can be created with adding \n. Please note: You have to include this inside the quotation signs.input("Choose a level of difficulty:\n 1) easy\n 2) medium\n 3) difficult\n") Choose a level of difficulty: 1) easy 2) medium 3) difficult

Concatenating strings: You can concatenate strings with variables using a plus or a comma. Please note: a comma will always add a space in the output. And if the variable is a float or an integer you have to convert the variable to a string first.language = 'Python' times = 4 print("I am learning "+language,str(times),"times a week.") → I am learning Python 4 times a week.

You can separate text in the terminal by adding empty lines using print() with empty brackets. But you can also define how to separate words, letters and general output using "sep" which inserts a string between values (by default a space) or "end" which appends a string after the last value (by default a new line):for i in range(2):   for j in range(2):     print("*") → *   *   *   * for i in range(2):   for j in range(2):     print("*", end="") → **** for i in range(2):   for j in range(2):     print("*", end=" ")   print() → * *   * *

F-strings help you add variables to strings in a quick manner. There are three different methods to format strings: Using %, str.format() or f-string. Remember to use different quotes if the string itself also contains quotes otherwise you will get an error.name = "Tim" occupation = "gardener" print(f"Hi I am {name} and I am a {occupation}.") print("Hi I am {} and I am a {}.".format(name, occupation)) print("Hi I am {1} and I am a {0}.".format(occupation, name)) print("Hi I am %s and I am a %s." %(name, occupation))   → Hi I am Tim and I am a gardener.

You can use ANSI code style to make your text more readable and creative. Please note: It highly depends on the type of terminal you are using. To print coloured or bold text in the terminal with ANSI escape codes:The format is:   \033[ = Escape code   32 = Text colour (32 for Yellow; 91 for Red; 95 for Purple)   \033[00m = ending formular   - Do not forget the quotation signs! - print("\033[91m This text is printed in Red.\033[00m") print("\033[95m This text is printed in Purple.\033[00m") print("\033[32m This text is Yellow.\033[00m") print("\033[1m"+"This text is bold."+"\033[0m")

Python provides a number of built-in methods to modify strings; e.g. count() returns the number of times a specified value occurs in a string; islower() returns True if all characters in the string are lower case and lower() converts a string into lower case. You can also split strings:string = "John" splitted_string = string.split('o')  → you can choose whichever marker you want print(splitted_string)  → ['J', 'hn']

Lists

Lists represent a collection of elements which can be of different data types. Other programming languages might refer to them as arrays. You can create a list in Python in just the same way as any other variable: by giving it a name and assigning it a value.

Lists are ORDERED and INDEXED, they can contain DUPLICATES and they are MUTABLE.animals = ['cat', 'dog', 'fish', 'bison']  → a list of strings numbers = [1, 7, 34, 20, 12]  → a list of integers my_list = []  → an empty list nested_list = [[1, 7, 34], ['cat', 'dog', 'fish'], [orange, blue, red]]  → a list of lists Each element inside a list has an index, starting from 0:cities = ['Berlin', 'Munich', 'Paris'] index         0         1         2The city on the position 0 is Berlin, on position 1 there is Munich and on position 2 there’s Paris.

How to work with lists using built-in methods:
To access or find an element by its index use square brackets containing the index:cities[1]  → Munich nested_list[1][1]  → dogTo find on the other hand the index of a certain element use the index-method: "list.index(value)". Please note: If the value appears more than once, you will only get the index of the first one and you'll get an error if the value is not included in the list.cities.index("Munich")  → 1

To insert a value at a particular position of a list use the insert-method: "list.insert(index, value)":numbers.insert(0, 45)  → to insert 45 at the beginning of the list of numbers

To add a value at the end of a list use the append-method:citites_to_visit = [] citites_to_visit.append('New York') citites_to_visit.append('Tokio') citites_to_visit.append('Copenhagen') print(citites_to_visit)  → ['New York', 'Tokio', 'Copenhagen']

To remove an element use the method with "list.remove(value)". Please note: if the value appears more than once, only the first one will be removed. The delete-method on the other hand needs to be fed with the index number instead of the value itself:cities_to_visit.remove("Tokio") del cities_to_visit[0] print(citites_to_visit)  → ['Copenhagen']

To sort a list of strings alphabetically or a list of numbers in ascending order you can use the following method:my_list = [5, 2, 30, 10, 4, 12, 46] my_list.sort() → [2, 4, 5, 10, 12, 30, 46]

To reverse a list of items:my_list.reverse() print(my_list)  → [46, 30, 12, 10, 5, 4, 2]

To slice an index means to access or to operate a certain action only on a fragment of a list. It follows the syntax of "list[starting index : ending index : slicing step]". The ending index is always excluded though. To access items from index [2] to index [3] in a list, you have to slice "list[2:4]". The slicing step is optional and automatically set to 1.my_list = [2, 4, 5, 10, 12, 30, 46] my_list[:]  → [2, 4, 5, 10, 12, 30, 46] all items my_list[2:4]  → [5, 10] items with index [2] and [3] my_list[-5:-2]  → [5, 10, 12] begins at the fifth and stops at the third item counting down from the end of the list (excluding the penultimate item)
my_list[:2]  → [2, 4] first two items my_list[3:]  → [10, 12, 30, 46] all items starting with the fourth (remember: the first item's index is 0)
my_list[2:7:2]  → [5, 12, 46] every second item, starting with the third and ending at index [6]
my_list[:-1]  → [46] last item my_list[::-1]  → [46, 30, 12, 10, 5, 4, 2] list in reverse order

Dictionaries

A dictionary is also a collection of items but with each element representing a unique "key: value" pair. Dictionaries are ordered and indexed (but only since Python version 3.7) and they are MUTABLE, which means that the stored data can be changed later on. You have to use a coma to seperate each data pair and curly brackets to wrap the whole thing.

Please note: Dictionaries cannot contain duplicate key values.my_dict = {     "brand": "Ford",     "model": "Mustang",     "year": 1964,     "year": 2020   → this line overwrites the previous one using the same key }

How to work with dictionaries using built-in methods:
With a loop you can iterate through all items inside a dictionary. Take for example one where key is a person's name and the corresponding value is the person's phone number  - resulting in "name : phone number pairs". With the items-method you can access the keys and the corresponding value individually:phonebook = {     "John" : 938477566,     "Jack" : 938377264 } for name, number in phonebook.items():   print("Phone number of " + name + " is " + str(number)) → Phone number of John is 938477566 → Phone number of Jack is 938377264 To see all keys of a dictionary as a list use the keys-method:print(phonebook.keys()) → ['John', 'Jack'] To see all corresponding values of a dictionary as a list use "dictionary.values()". And to only get the value of a specific key use the get-method with "dictionary.get(key)":print(phonebook.values()) → [938477566, 938377264] print(phonebook.get("John") → 938477566 To remove a certain key:value-pair from a dictionary use "dictionary.pop(key)" or delete as in "del dictionary[key]". To remove all elements from a dictionary use "dictionary.clear()".del phonebook['John']

Tuples

Python has another sequence type which is called tuple. Once a tuple is created, you cannot change its values, it is IMMUTABLE. We define a tuple by putting a comma-separated list of values inside round brackets: "tuple = (value, value, value)" weekdays = ('Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday', 'Sunday') mixed_data = ("Orange", [10, 20, 30], (5, 15, 25))  → items can be of different data types print(mixed_data[1][1])  → 20 To let Python know that you want to create a tuple with just a single value you have to add a trailing comma:print((3,)) NOT print((3))

How to work with tuples using built-in methods:
To access a single element use "tuple[index]":print(weekdays[0])   → Monday print(weekdays[-1])   → Sunday You can also count values or look up an index inside a tuple:print(weekdays.count('Monday'))   → 1 print(weekdays.index('Wednesday'))  → 2 print(len(mixed_data))  → 3 And you can even join tuples: tuple3 = tuple1 + tuple2joined_tuple = weekdays + mixed_data print(joined_tuple) → ('Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday', 'Sunday', 'Orange', [10, 20, 30], (5, 15, 25))

You CANNOT assign, insert or append a new value because a tuple is immutable. There is a workaround though: First you convert the tuple into a list, then change the list, and convert the list back into a tuple.x = ("apple", "banana", "cherry") y = list(x) y[1] = "kiwi" x = tuple(y) print(x)  → ('apple', 'kiwi', 'cherry')

Sets

A "Set" is another sequence data type in Python written with curly brackets. Set items are also immutable - except for removal and addition. They are unordered, therefore the items can appear in a different order every time. A set cannot accept identical items (like a dictionary) and items cannot be referred to by index or key.
